# Distributed Systems Architecture Capstone & Benchmarking Sandbox

A comprehensive system testing dashboard built to validate, stress-test, and benchmark advanced distributed systems design patterns under high concurrency and simulated infrastructure drops.

## Architectural Capabilities Verified
* **Fault Injection Simulations**: Tests centralized vs decentralized error boundaries under systematic network drops.
* **Optimistic Concurrency Control (OCC)**: Validates transaction version checking structures to prevent race conditions during simultaneous writes.
* **Snapshot Acceleration Metrics**: Measures the recovery speed advantages achieved by using periodic state snapshots instead of raw history replays.
